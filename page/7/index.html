Attention is all you need

2023-02-19
One minute
Attention is all you need “Attention is all you need” is a research paper that proposes a new way of making an AI. Attention is mapping a query and a set of key-value pairs to an output, all of which are vectors: the query, the keys, and the values. The results are calculated as a weighted sum of the values, ith each value’s weight determined by the query’s compatibility function with its corresponding key.
